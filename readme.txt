目錄結構：
/defulan-baseball
  ├── index.html   (首頁)
  ├── style.css    (全站與首頁共用樣式)
  ├── main.js      (全站共用動態腳本)
  └── /images      (請自行建立此資料夾存放圖片)

圖片替換指南：
目前網頁中的圖片使用了預留位置 (Placeholder)，例如：
<img src="https://via.placeholder.com/1920x800/2D4263/FFFFFF?text=Hero+Image" alt="高海拔 800 公尺的操場景觀與球員訓練的熱血大圖">

當您準備好真實的照片後，請將照片存入 /images 資料夾，並修改 HTML 中的 src 屬性，例如：
<img src="./images/hero-bg.jpg" alt="高海拔 800 公尺的操場景觀與球員訓練的熱血大圖">
